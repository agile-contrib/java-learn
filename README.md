## How to use?

首先，你需要进入文件夹 ```src``` 后执行编译命令：

```
javac XXX.java
```

如果出现编码错误，可以改用下面的命令进行编译：

```
javac -encoding UTF-8 .\XXX.java
```

编译成功后，执行下面命令即可运行：

```
java XXX
```
